반응 선택자

사용자의 움직임에 따라 동작을 하는 선택자

        사용자가 클릭한 경우:
        선택자:active{설정 내용}

        사용자가 마우스를 태그 위에 올려두는 경우:
        선택자:hover{설정 내용}

자손 선택자와 후손 선택자

            자손 선택자 : 선택자>자손선택자{설정 내용}
            -> 선택자의 바로 하위 레벨 요소(태그)를 지정

            후손 선택자 : 선택자 후손선택자{설정 내용}
            -> 선택자의 모든 하위 레벨 요소 지정

동위 선택자

동위 관계에서 뒤에 위치한 태그를 선택할 때 사용

            바로 뒤 요소 하나 선택
            선택자a + 선택자b{설정내용}

            뒤에 있는 모든 요소 선택
            선택자a ~ 선택자b{설정내용}

            사용 가능한 input 태그 선택
