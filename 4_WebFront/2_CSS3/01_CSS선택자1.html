CSS 기본 선택자

CSS란?

Cascading Style Sheets 약자로
HTML 같은 마크업 언어에 종속되어 레이아웃과 스타일을 정의하는 자유도 높은 마크업 언어.

선택자란?

특정한 HTML 태그를 선택할 때 사용하는 기능으로,
원하는 태그를 선택하여 스타일, 기능을 적용할 수 있음.
