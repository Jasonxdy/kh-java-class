속성 선택자

            
            요소[속성명 = 속성값] : 해당 속성명과 속성값이 같은 객체(요소) 선택. (완전히 일치)

            요소[속성명 ~= 속성값] : 속성값이 특정 단어를 포함한 객체 선택.
                -> 클래스가 여러 개인 요소 선택 시 자주 사용됨.
            
            요소[속성명 ^= 속성값] : 속성값이 특정 단어로 시작하는 객체 선택.
            
            요소[속성명 $= 속성값] : 속성값이 특정 단어로 끝나는 객체 선택.

            요소[속성명 *= 속성값] : 속성값이 특정 단어를 포함하고 있는 객체 선택. (일부)
