이미지 관련 태그

<img> : 웹 페이지에 사진이나 그림을 삽입할 때 사용하는 태그

            img태그 속성
            - src : 삽입할 이미지 경로를 지정하는 속성
            - alt : 이미지를 설명하는 텍스트를 추가하는 속성
                    이미지가 출력이 되지 않으면 표시됨.
            - width/height

src 속성

alt 속성

이미지의 경로가 잘못되거나, 이미지를 제대로 표현할 수 없는 경우
대체 텍스트를 표시하는 용도로 사용하는 속성.
+ 시각 장애인을 위한 ScreenReader에서 이미지를 대체하여
낭독해주는 텍스트를 작성.

width / heigth

태그의 크기(높이, 너비)를 지정하는 속성.
고정된 크기의 단위(px 등) 및 가변 크기 단위(% 등)로 지정 가능.

고정 크기 단위(px) : 화면 크기가 변경되어도 크기가 변하지 않음

가변 크기 단위(%) : 화면의 크기가 변경되면 기준 사이즈에 따라 크기가 변경됨
