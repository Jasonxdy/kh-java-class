Hello, Servlet world!

서블릿이란, 웹 서비스를 위한 자바 클래스를 말하며, 자바를 사용해 웹을 만들기 위해 필요한 기술로 javax.servlet.http.HttpServlet 클래스를 상속 받는다.
다시 말해 기존의 java 파일에 웹 페이지 구현을 위한 html이 들어간 구조라고 볼 수 있다.
서블릿은 클라이언트의 요청을 처리하고 그 결과를 HTML을 사용하여 요청에 대한 응답을 다시 클라이언트에게 전달하는 구현 규칙이 있다.
단, servlet에 작성된 HTML 코드 변경 시 재 컴파일 해야 한다는 단점이 있다.
